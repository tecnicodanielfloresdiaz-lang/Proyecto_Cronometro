⏱️ JavaScript Stopwatch

Un cronómetro digital moderno y funcional construido con Vanilla JavaScript. Este proyecto demuestra el control preciso del tiempo y la manipulación dinámica de la interfaz de usuario (DOM).

🚀 Funcionalidades

Control Total: Botones dinámicos para Iniciar, Pausar y Reiniciar el contador.
Precisión: Visualización de horas, minutos y segundos con formato de doble dígito (00:00:00).
Interfaz Dinámica: Actualización en tiempo real del DOM sin recargar la página.
Diseño Limpio: Estilizado con CSS3 para una experiencia de usuario intuitiva.

🛠️ Tecnologías Aplicadas
HTML5: Estructura del marcador y controles.
CSS3: Diseño responsivo y estados visuales de los botones (hovers/active).
JavaScript (ES6+):
setInterval() y clearInterval() para la gestión del tiempo.
Manipulación del DOM para inyectar los valores actualizados.
Lógica de formateo de texto (strings) para mantener el formato de reloj.
